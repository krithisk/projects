Minor Project: Zomato Dataset Analysis

Objective

To analyze restaurant data from Zomato and predict ratings using machine learning models. The goal was to identify key features influencing restaurant ratings and optimize prediction accuracy.

Techniques & Concepts Used

  • Data Cleaning: Handling nulls, duplicates
  • Feature Engineering: Correlation analysis, outlier treatment
  • Modeling: Random Forest, Linear Regression, Decision Tree
  • Evaluation: Accuracy, RMSE

Process Flow

Model Used

Random Forest was selected for its ensemble capability and robustness against overfitting. It provided high accuracy and interpretability for feature importance.

Result

Achieved strong predictive performance and published findings at ICDEIS'22. Key insights included city-wise restaurant density and pricing trends.

Internship Project: Regression Modeling for Marketing Optimization

Objective

To build regression models that measure the impact of marketing channels on revenue for a pharmaceutical client. The goal was to optimize budget allocation and identify high-performing channels.

Techniques & Concepts Used

  • EDA: Monthly aggregation, hypothesis testing
  • Feature Engineering: Permutation importance, correlation analysis
  • Models: Random Forest, AdaBoost, Gradient Boost, LightGBM
  • Deployment: Co.dx platform, Azure Data Lake

Process Flow

Model Used

LightGBM was selected for deployment due to its superior performance on non-linear relationships and high accuracy (78.1%). Hyperparameter tuning was done using GridSearchCV and Optuna.
